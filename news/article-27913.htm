Vue2.0中消除警告方法：eslint-plugin-vuefix

消除vue2.0版本中的警告方法：

①package.json文件中：（更改lint字段）

"lint":"eslint --fix --ext .js,.vue src",

②.eslintrc.js文件中：（更改plugins字段）

 plugins:['vuefix'],

③ 下载eslint-plugin-vuefix插件：

$ npm i eslint-plugin-vuefix-D

④ //运行命可：

$ npm run lint

⑤ //运行项目：

$ npm run dev
